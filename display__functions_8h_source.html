Wordle 1
The Wordle Game is a console-based implementation of the popular word puzzle game, Wordle. The objective of the game is for the player to guess a hidden 5-letter word within a limited number of attempts. After each guess, the game provides feedback on each letter's accuracy—whether it's in the correct position, present but in the wrong position, or not in the word at all—using color-coded visual cues. The game is designed to be engaging and educational, offering players a fun way to improve their vocabulary and logical thinking skills.
